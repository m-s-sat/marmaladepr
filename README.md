# NOIRE Restaurant Website

A modern, elegant website for NOIRE - an intimate fine dining restaurant offering an immersive culinary experience.

## Features

- Modern, responsive design
- Interactive menu sections
- Reservation functionality
- Contact information

## Tech Stack

- **React** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Shadcn/UI** - Component library

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The site will be available at `http://localhost:8080`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── components/     # Reusable UI components
├── data/          # Restaurant data and menu items
├── pages/         # Page components
└── lib/           # Utility functions
```
